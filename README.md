Smart Shopping Cart with Automatic Billing
IoT-based smart shopping cart that automates item detection and billing using RFID and ESP8266. The system enables real-time bill calculation, item tracking, and cashier-less checkout, reducing manual effort and checkout time in retail stores.
Tech Stack
  ESP8266, RC522 RFID Reader
  OLED Display, RFID Tags
  Embedded C/C++, Arduino IDE
  Blynk IoT Platform
Key Features
  Automatic item scanning using RFID
  Real-time bill updates
  Add/remove item support
  OLED and mobile app display
  Cashless self-checkout
Outcome
Eliminates barcode scanning, reduces billing errors, and improves shopping efficiency using IoT automation.
Project Type
Academic Mini Project (IoT & Embedded Systems)
